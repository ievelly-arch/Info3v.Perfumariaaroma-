# Geração de Relatórios 

**COMO Gerente do Sistema,**
**QUERO visualizar relatórios de vendas e estoque**
**PARA analisar o desempenho da perfumaria e tomar decisões baseadas em dados.**

### *Critérios de Aceitação:*

* O sistema deve gerar relatórios de produtos mais vendidos.
* Deve ser possível visualizar o faturamento em um determinado período.
* O sistema deve apresentar a situação atual do estoque.
* Os relatórios devem ser organizados de forma clara e de fácil compreensão.
