# Registro de vendas 

**COMO Operador do Sistema**
**QUERO registrar as vendas realizadas na perfumaria**
**PARA manter o controle das transações e acompanhar o desempenho das vendas.**

### *Critérios de Aceitação:*

* O sistema deve permitir selecionar os produtos vendidos e suas respectivas quantidades.
* O sistema deve calcular automaticamente o valor total da venda.
* O sistema deve atualizar o estoque dos produtos após a conclusão da venda.
* Uma venda só pode ser registrada se os produtos estiverem previamente cadastrados no sistema.
