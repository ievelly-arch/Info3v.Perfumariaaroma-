# Consulta e busca de produtos 

**COMO Operador do Sistema,**
**QUERO pesquisar produtos por nome, marca ou categoria**
**PARA localizar rapidamente itens no sistema e facilitar o atendimento ao cliente.**

### *Critérios de Aceitação:*

* Deve existir um campo de busca que permita pesquisar pelo nome e marca dos produtos.
* O sistema deve permitir filtrar os resultados por categoria.
* A lista de resultados deve exibir informações como preço e quantidade disponível em estoque.
