# Cadastro de produtos 
**COMO** Operador do Sistema e administradora,
**QUERO** cadastrar os produtos da perfumaria
**PARA** que seja possível organizar, controlar e disponibilizar os itens para venda.

### **Critérios de Aceitação:**

* Deve ser possível informar nome do produto, marca, categoria, preço, quantidade em estoque e descrição.
* O sistema deve permitir o cadastro de diferentes categorias de produtos (ex: perfumes, hidratantes, kits).
* O sistema deve impedir o cadastro de produtos duplicados com o mesmo nome e marca.
* Deve ser possível editar ou inativar um produto (ex: produto fora de linha ou indisponível).
