# Controle de estoque 

**COMO** Operador do Sistema,
**QUERO** acompanhar e atualizar o estoque dos produtos
**PARA** garantir a disponibilidade dos itens e evitar faltas ou excessos.

### *Critérios de Aceitação:*

* O sistema deve atualizar automaticamente a quantidade em estoque após cada venda ou reposição.
* Deve ser possível visualizar a quantidade atual de cada produto em estoque.
* O sistema deve alertar quando um produto estiver com estoque baixo.
* Deve ser possível registrar entradas de novos produtos no estoque.
