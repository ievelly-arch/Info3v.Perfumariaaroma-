# Perfumariaaroma
## Uma Breve Descrição do Contexto 
#Repositório criado para o gerenciamento do desenvolvimento de um sistema de gestão de perfumaria, com foco em controle de estoque, vendas e clientes.
#A Perfumaria aroma, pertencente a uma holding familiar com atuação em diversas empresas ao redor do mundo, busca otimizar seus processos internos por meio de uma solução tecnológica. O objetivo é tornar as atividades relacionadas ao controle de produtos, estoque, vendas e clientes mais ágeis, organizadas e eficientes, garantindo que as decisões gerenciais sejam baseadas em dados concretos. A partir da análise das necessidades do negócio, propõe-se o desenvolvimento de um sistema capaz de acompanhar todas as etapas da perfumaria, desde o cadastro e entrada de produtos no estoque até a finalização das vendas, registrando informações essenciais para melhorar a gestão e apoiar a tomada de decisões.
